馮紐曼模式:此架構主要在於 **儲存模式**

馮紐曼模式有4大子系統

1. 記憶體(Memory)
   - 同時儲存 **程式** 和 **資料**

2. 算術邏輯單元(ALU)
   - 負責資料的運算處理
 
3. 控制單元(CU)
   - 控制記憶體，輸入輸出及算術邏輯單元的運作

4. 輸入/輸出(I/O)
   - 廣義的輸出入子系統還包括 **次要的儲存設備**
